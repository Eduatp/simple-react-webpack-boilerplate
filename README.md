Simple React, webpack boilerplate
====================================

It doesn't get much simpler than this. Spin up your front-end environment with ease.

## Setup

* `npm install`

* `npm start`

* `open index.html`

## What's inside

````
$ tree --charset unicode
.
|-- LICENSE
|-- README.md
|-- index.html
|-- package.json
|-- scripts
|   |-- index.js
|   `-- welcome-message.js
`-- webpack.config.js
````

Yep, that's it.
